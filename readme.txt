== Description ==
MonacaPress plug-in is WordPress app development support tool.
And source code to make the application that runs on iOS and Android,
It provides a mechanism for cooperation mobile application development platform as "Monaca".
You can build to native apps and editing of source code, and set to give the app market in Monaca.
In addition, because of the JS and HTML5, source code can be customized in Web technologies familiar.

== Installation ==
1. Upload MonacaPress to  /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
